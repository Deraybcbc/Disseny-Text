Text-decoration

se usa para establecer el formato de texto a subrayado ( underline ) y suprarrayado ( overline ), tachado ( line-through ) o parpadeo ( blink ).

Word-spacing

define una cantidad adicional de espacio entre palabras. El valor debe estar en formato de longitud; no se permiten valores negativos.

Text-align-last

establece la alineación horizontal del contenido a nivel de línea dentro de un elemento de bloque o caja de celda-tabla. Esto significa que funciona como vertical-align pero en dirección horizontal.

Overflow-wrap

permite especificar que el navegador puede romper una línea de texto dentro del elemento dirigido en múltiples líneas en un lugar inquebrantable.

Line-break

produce un salto de línea en el texto (retorno de carro).
